# Code of Conduct

This project aims to be a respectful and constructive space.

## Expected Behavior

- Be respectful and patient.
- Give clear technical feedback.
- Assume good intent.
- Keep discussions focused on the project.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information.
- Spam or repeated off-topic comments.
- Aggressive demands for unpaid work.

Maintainers may edit, hide, or remove comments and may block users who repeatedly violate these expectations.
