# Security Policy

## Supported Versions

Security fixes are provided for the latest tagged release.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security problems.

Use GitHub Security Advisories:

```text
https://github.com/kadonix/laravel-swagger-routebook/security/advisories/new
```

Include:

- affected version
- Laravel and PHP versions
- reproduction steps
- impact
- any suggested fix

We will review the report and coordinate a fix before public disclosure.
